# FUTURE_DS_02

Customer Churn Analysis Dashboard

📌 Project Overview

This project analyzes customer churn behavior using business analytics techniques.
The goal is to identify patterns and factors that influence customer retention and provide actionable business insights.

🛠 Tools Used
Microsoft Excel (Data Cleaning)
Power BI (Data Visualization & Dashboard)

📊 Key Features
Customer churn distribution analysis
Monthly charges comparison (churn vs non-churn)
Contract type impact on churn
Internet service analysis
Customer tenure insights
Interactive filters (slicers)

📈 Key Insights
Month-to-month contracts have the highest churn
Higher monthly charges increase churn risk
New customers are more likely to churn
Fiber optic users show higher churn
Long-term customers are more loyal
